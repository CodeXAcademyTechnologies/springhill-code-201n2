Audio Video Elements

Audio

Common attributes to control playback of audio

  • Autoplay: boolean
  • Controls: provides playback controls to the user
  • loop: boolean. plays the sound over and over and over and over and over
  • muted: boolean. can be used to mute audio on page load
  • src: link to the audio embeds the audio to the source code in HTML
  • Volume: range of 0.0 to 1.0 silent to loudest

Video

Common attributes of video Elements

  • autoplay: boolean to auto play video
  • controls: gives user control of video like playing the video
  • height: set the hight of the vide. must be an absolute value. no percentages
  • muted: boolean: mutes on page load
  • poster: use an alternate image in place of the first frame before playback has initialized
  • src: path to the video to embed into source code
  • volume: range of 0.0 to 1.0 silent to loudest
  • width: may be an absolute value or a percentage
